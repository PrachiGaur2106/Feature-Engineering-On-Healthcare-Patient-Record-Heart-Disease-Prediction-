# Feature Engineering on Healthcare Patient Records

**Name:** Prachi Gaur

---

## Project Description

This project focuses on applying **feature engineering techniques** on healthcare patient records to prepare the data for machine learning analysis. Feature engineering helps transform raw medical data into meaningful features that improve model performance and understanding of health patterns.

The project analyzes patient health parameters and prepares the dataset so that machine learning models can effectively predict possible health risks such as **heart disease**.

---

## Objective

The main objective of this project is to apply **feature engineering techniques** on healthcare data to:

- Improve data quality  
- Create useful features  
- Prepare the dataset for machine learning models  

---

## Dataset

The dataset used in this project contains healthcare patient records with various medical attributes such as:

- Age  
- Sex  
- Chest Pain Type  
- Resting Blood Pressure  
- Cholesterol  
- Fasting Blood Sugar  
- Maximum Heart Rate  
- Exercise Angina  
- Oldpeak  
- ST Slope  

### Target Variable

**HeartDisease**

- `0` → No Heart Disease  
- `1` → Heart Disease

## Feature Engineering Steps

The following feature engineering techniques were applied:

### 1. Data Cleaning
Checking for missing or incorrect values.

### 2. Encoding Categorical Data
Converting text values like gender into numerical format.

### 3. Creating New Features
Generating additional features such as **Age Group** and **High Risk Indicator**.

### 4. Correlation Analysis
Using a **heatmap** to analyze relationships between different features.

### 5. Feature Selection
Identifying important variables that influence heart disease prediction.

## Results

Feature engineering helped in:

- Improving dataset quality  
- Identifying important medical features related to heart disease  

The processed dataset can be effectively used for **machine learning models** to make accurate predictions.

Feature engineering plays an important role in **machine learning projects**, especially in healthcare data analysis. Properly engineered features help improve **model accuracy** and provide better insights into **patient health conditions**.
